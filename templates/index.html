Persistencia de datos - Semana 12

Este proyecto implementa dos mecanismos de persistencia según la consigna:

  • Base de datos SQLite con SQLAlchemy: Utilizado para la gestión de clientes, productos y pedidos.
  • Archivos (TXT, JSON, CSV): Accede a la sección Datos para probar el formulario que guarda y lee información en estos formatos, cumpliendo con el requisito de la semana 12.

Los archivos generados se almacenan en la carpeta inventario/data/ del proyecto.
